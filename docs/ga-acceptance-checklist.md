## A. 构建与门禁

- [ ] `npm run prebuild:release` 全部 PASS（含 `POST /api/analyze` fast）
- [ ] NSIS 安装包产出：`CNexus 2.0_2.4.0_x64-setup.exe`
- [ ] 安装包内含 `resources/help/cnexus-personal-troubleshooting.md`

---

## B. 安装 / 升级（B2）

- [ ] **干净安装**：无旧 CNexus 进程，安装成功，桌面快捷方式可启动
- [ ] **覆盖升级**：旧版占 7864 时，安装器预检提示 → 确认后继续 → 启动正常
- [ ] 确认无残留 **:8000** 杀进程逻辑误杀（仅 7864/3000）

---

## C. 网关生命周期（C1 + C2）

| 步骤 | 操作 | 期望 |
|------|------|------|
| 1 | 正常启动桌面版 | 无红色横幅；`/api/status` 正常 |
| 2 | 任务管理器结束 `cnexus-runtime` | 顶部出现 **本地网关未连接** 横幅 |
| 3 | 点 **重启网关** | 10s 内横幅消失，功能恢复 |
| 4 | 点 **我知道了** | 横幅静默 30 分钟 |
| 5 | 点 **查看自救指南** | 应用内 Help 模态打开，内容可滚动 |
| 6 | 关于 → **打开日志目录** | 资源管理器打开 `%LOCALAPPDATA%\CNexus\data` |

---

## D. 分析降级（C5 + C2）

- [ ] 停止 Ollama 或使用不可用模型 → 思考 tab **深度思考** 触发降级
- [ ] UI 显示黄色/蓝色 warning，**非红色致命错误**
- [ ] 全局横幅显示 **分析已降级** + 自救指南入口
- [ ] 恢复 Ollama 后重试，深度分析恢复正常

---

## E. 多入口访问

- [ ] **Tauri 浮窗** `tauri.localhost`：API 走 `127.0.0.1:7864`
- [ ] **本机浏览器** `http://127.0.0.1:7864/shell`：数据与 API 正常
- [ ] **手机 LAN** `http://<PC-IP>:7864`：页面 + API 可用（防火墙已放行）

---

## F. 离线帮助

- [ ] 关于对话框可打开日志目录（Tauri）
