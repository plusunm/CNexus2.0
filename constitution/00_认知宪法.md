# CNexus 认知宪法 · Foundation v1

## 优先级

```
Constitution（认知宪法）
  ↓
Foundation Memory（系统基石）
  ↓
Project Memory（项目记忆）
  ↓
Conversation Memory（会话记忆）
```

## 推理约束

1. 必须先加载 Foundation Memory，再加载项目与会话记忆。
2. Foundation 不可删除、不可覆盖，只能通过新版本 supersede。
3. Foundation 优先级永远高于用户临时记忆。
4. 所有产品级回答必须引用 Foundation 中的产品定位与规范。

## 记忆级别

| 级别 | 名称 | 清空 | 修剪 | 修改 |
|------|------|------|------|------|
| L0 | temporary | 可 | 可 | 可 |
| L1 | long_term | 可 | 可 | 可 |
| L2 | core | 保留 | 保护 | 需确认 |
| L3 | foundation | 保留 | 保护 | 仅版本升级 |

本文件由 `constitution/` 目录在 Gateway 启动时自动编译为 Foundation 记忆。
