# CNexus 2.0 Personal Edition — v2.3.0 发布说明

> 面向第一次使用的小白用户：这版能做什么、怎么开始、和上一版比改了什么。

**发布日期：** 2026-06-22  
**下载：** [GitHub Releases — v2.3.0](https://github.com/plusunm/CNexus2.0/releases/tag/v2.3.0)

---

## 一句话介绍

**CNexus 2.0 是你电脑上的「第二大脑」**：可以聊天、上传文档、记住重要内容，并在下次对话时自动把相关记忆拿出来参考——数据默认都在本机，不上传云端。

---

## 3 分钟上手

1. **安装 Python 3.10+**（若还没有）  
2. **双击** 项目里的 `start_cnexus.bat`  
3. 浏览器会自动打开 **http://127.0.0.1:7864**  
4. 左侧选 **「对话」** 开始聊天；选 **「上传记忆」** 导入 PDF / Word / 文本  

想更聪明地回答？到 **「模型」** 页填入 DeepSeek 或 OpenAI 的 API Key，或在本机安装 [Ollama](https://ollama.com) 跑本地模型。

---

## 本版新功能（用人话讲）

### 1. 聊天更像「助手」，不再复读你的话

以前若没配好 AI 模型，系统可能把你发的「你好」原样复读回来。  
**现在：**

- 会正常介绍自己、提示你如何配置模型  
- 若已配置 DeepSeek / Ollama，会优先用外部 AI 回答  
- 输入框上方可选 **记忆范围**：本机 / 组群 / 全网（决定参考哪里的记忆）

### 2. 对话设置更清晰

- **快速 / 深度 / 不参考记忆**：控制回答速度和是否注入记忆  
- **准确 / 深入思考**：控制回答风格  
- **记忆范围**：单独放在输入框上方，不再和「对话偏好」重复  

### 3. 上传文档更稳、更快

- 修复了单文件上传卡住十几秒的问题  
- 支持批量上传并看进度  
- 上传入口：**Second Brain → 上传记忆**

### 4. 聊天窗口不再把页面顶出屏幕

- 消息多了会在**聊天区域内滚动**，输入框始终固定在下方  

### 5. 界面源码一并开源

- `frontend/` 目录包含完整前端工程，可自行修改并执行 `npm run build:personal` 重建界面

## 常见问题

**Q：一定要联网吗？**  
不上网也能用基础功能；要用云端 AI（DeepSeek 等）需要网络和 API Key。

**Q：我的文档会上传到 GitHub 吗？**  
不会。记忆存在本机 `data/` 文件夹，已在 `.gitignore` 中排除，不会随代码推送。

**Q：Clash / 代理软件导致连不上 AI？**  
请关闭失效的系统代理，或在 Clash Verge 里关掉「系统代理」后再试。

**Q：怎么完全退出？**  
运行 `stop_cnexus.bat`，或在任务管理器结束 `python app_v2.py`。
