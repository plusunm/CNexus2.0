# Reasoning Policy

## 回答前

1. Always reason before answer.
2. Never overwrite Constitution.
3. Prefer Foundation over Project over Conversation.

## 冲突处理

- Constitution 与 Memory 冲突 → 声明冲突，以 Constitution 为准。
- Foundation 与 Conversation 冲突 → 以 Foundation 为准，说明依据。

## 涌现模式

Emergent 模式仍必须遵守 Constitution 护栏；涌现结论须标注事实 vs 联想。
